# Active Directory Enterprise Lab

## Objective
Build and secure a Windows domain environment that simulates a small organization.

## Skills Demonstrated
- Active Directory administration
- DNS and DHCP configuration
- Group Policy management
- User onboarding and offboarding
- Access control
- Windows Server administration

## Tools Used
- Windows Server 2022
- Windows 10/11
- Active Directory Domain Services
- Group Policy Management
- PowerShell

## Lab Steps
1. Installed Windows Server.
2. Promoted server to a domain controller.
3. Created Organizational Units.
4. Created users and security groups.
5. Configured password and lockout policies.
6. Applied Group Policy Objects.
7. Tested login and access permissions.
